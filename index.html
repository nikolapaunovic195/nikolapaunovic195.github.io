Privacy Policy - Watchfaces by Nikola Paunovic

To use some or all feautres of this app, the user may need to allow requested permissions. These permissions are only used for the watchface to be able to display information in its complications.
Any data processed by the app never gets collected nor stored, and is only used to provide information when the watchface is in use.
This data never gets shared with anyone, and is never used for advertising purposes.
The apps can only access data allowed by the user through a permission request dialog.
